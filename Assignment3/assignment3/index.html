What is HTML5?

HTML is the standard markup language used for creating web pages on the World Wide Web. HTML stands for Hyper Text Markup Language and describes the structure of websites using markup. Furthermore, HTML elements are the building blocks for web pages and these elements are represented by HTML tags.
